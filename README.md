# 속도가, 신뢰입니다 — AX 특강 웹앱

메모리 품질실 CS팀 AX 특강 발표 자료입니다. 대강당(1,000명+) 발표용 슬라이드형 웹앱으로, 단일 HTML 파일 하나로 동작합니다.

**작성자**: 메모리 품질실 CS팀 · 민희순 PL

---

## 특징

- **의존성 0** — 외부 라이브러리·CDN·폰트 로드가 전혀 없어, 인터넷이 없는 사내망에서도 완전히 작동합니다.
- **단일 파일** — `index.html` 하나가 전부입니다. 저장소에 올리거나 파일을 그대로 열어도 됩니다.
- **슬라이드형 17장면** — 프리젠터 리모컨/키보드로 넘기며, 장면 안에서 단계별 애니메이션이 재생됩니다.

## 발표 조작키

| 키 | 동작 |
|---|---|
| → / PageDown / Space / Enter / 클릭 | 다음 (단계 → 다음 장면) |
| ← / PageUp | 이전 장면 |
| Esc | 현재 장면의 모든 단계 즉시 표시 (연출이 꼬였을 때 안전장치) |
| Home / End | 처음 / 마지막 장면 |
| ? | 조작키 도움말 열기·닫기 |
| 숫자 1 / 2 | (투표 장면) 현장 거수 집계 시작 — 1번 먼저, 2번 그다음 |

## 로컬에서 열기

파일을 더블클릭해 브라우저로 열면 됩니다. 발표 시에는 F11로 전체 화면을 권장합니다.

## GitHub Pages 배포

1. 이 저장소를 GitHub에 올립니다.
2. Settings → Pages → Source: **Deploy from a branch** → Branch: `main`, 폴더: `/ (root)` → Save
3. 1~2분 뒤 `https://<계정>.github.io/<저장소명>/` 에서 열립니다.

`index.html`이 루트에 있어야 주소 진입 시 바로 열립니다.

## 사내 Git 이관

GitHub에서 소스를 내려받아(Code → Download ZIP) 사내 Git 저장소에 동일하게 커밋합니다. 외부 의존이 없으므로 사내망에서도 그대로 작동하며, 사내 정적 호스팅(Pages류)을 지원하면 활성화해 링크로 공유할 수 있습니다.
